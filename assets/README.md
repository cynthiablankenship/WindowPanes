# WindowPanes Icon Assets

Place production icon assets here:

- `icon.svg`: Source vector icon for editing and regenerating raster assets.
- `icon.ico`: Windows app, window, and taskbar icon. Preferred on Windows.
- `icon.png`: Cross-platform fallback icon, generated from the source artwork.

Art direction: a clean blue glass pane or layered panes, modern and slightly glossy, with a strong square-friendly silhouette and no text.
